Magnetic Navigation

Magnetic navigation is a way to obtain a position on Earth without GPS. Moreover, this navigation system is passive and cannot be jammed. However, there is only one problem: the disturbance of the measurements by the carrier.
There is currently the "Tolles-Lawson" method, but it is insufficient for onboard magnetometers. My work therefore aimed to study compensations using different artificial intelligence models. This work has shown that neural networks can improve the performance of compensation compared to Tolles-Lawson up to 50% on some sections of flight mainly through a convolutional network.

Magnetic Navigation

Magnetic navigation is a way to obtain a position on Earth without GPS. Moreover, this navigation system is passive and cannot be jammed. However, there is only one problem: the disturbance of the measurements by the carrier.
There is currently the "Tolles-Lawson" method, but it is insufficient for onboard magnetometers. My work therefore aimed to study compensations using different artificial intelligence models. This work has shown that neural networks can improve the performance of compensation compared to Tolles-Lawson up to 50% on some sections of flight mainly through a convolutional network.

For more information a link to Github with the complete code and the report on this subject are available.

Rain Gauge application

Recently I saw my father taking rain readings in a small notebook. When I opened the notebook I saw a large number of records. As a data scientist, it seemed a shame not to use all this data.
So I thought it would be interesting to create an application (under android because that's what my father and I use) to save all this data but also to have some info on this data like the average rain value per month. Moreover it would allow to save the rain records in a more secure way.

Clicking on the icon launches the download of the application's apk.
